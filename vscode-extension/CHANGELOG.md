# Change Log

All notable changes to the "kimi-ecosystem" extension will be documented in this file.

## [0.1.0] - 2024-01-01

### Added
- Initial release
- Command palette integration for kimi-audit, kimi-admin, kimi-converge
- Tree view showing convergence loop status
- Output channel for real-time logs
- Configuration UI for YAML files
- Problem matchers for security findings
- Code lenses for running audits on open files
- Keyboard shortcuts (Ctrl+Shift+K A for audit, Ctrl+Shift+K C for converge)
- Auto-run audit on save option
- Security diagnostics provider with severity mapping
